About the Jam

An open-to-all digital event featuring some jedi-level code poets and their work in a performance format.
If you'd like to join the band of featured poets, reach out to us with your code poem before ____.
Here's the playbook:

  • You can use any computer programming language
  • Must compile. Must run.
  • Possibilities (∞) : Poetry in Spoken Code, Interactive Code Poetry, Code Poem with Speech Synthesis, Cyborg Poetry, Visual Poetry
  • Submit your poem here. It can be a text file, a screenshot/image or an audio/video link
