Code Poetry Jam

Java Sonnets, C++ Ballads and Python Haikus? Why not!
Poetry + Coding = The New Avant-Garde
India’s first poetry jam featuring poems written in computer code, performed by machines and people.

An open-to-all digital event featuring some jedi-level code poets and their work in a performance format.
If you'd like to join the band of featured poets, reach out to us with your code poem before ____. Here's the playbook:

  • Any computer programming language
  • Must compile. Must run.
  • Possibilities (∞) : Poetry in Spoken Code, Interactive Code Poetry, Code Poem with Speech Synthesis, Cyborg Poetry, Visual Poetry
